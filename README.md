# IndiaPay 🇮🇳💳

IndiaPay is an ultra-premium, enterprise-grade fintech web application built with a focus on cutting-edge design, flawless performance, and advanced layout architectures. It serves as a comprehensive dashboard for digital payments, wallet management, analytics, and active security tracking.

## ✨ Features

- **Flagship UI/UX Framework:** High-contrast dark mode, deep glassmorphism components, and smooth GPU-accelerated micro-interactions (operating without heavy JS animation frameworks).
- **Interactive 3D Virtual Cards:** Beautifully integrated interactive card flip functionality featuring simulated holographic glares.
- **Global Data State Syncing:** Powered by a centralized internal database simulate (`db.ts`) accurately mapping transaction ledgers securely across the application ecosystem.
- **Flawless Responsive Grids:** Natively shifting CSS grid layouts transitioning organically between Mobile (1 column array), Tablet (2 column grids), and Desktop (3 column masonry) environments.
- **Rich Dashboard Tooling:** Fully functional active Loan progress bars, dynamic Utility Bill lists that natively execute mock API POST sequences, and real-time comparitive income/expense Analytics visuals.

## 🚀 Quick Start

1. Install Dependencies: 
```bash
npm install
```

2. Start Development Server: 
```bash
npm run dev
```

Build Process:
```bash
npm run build
```
